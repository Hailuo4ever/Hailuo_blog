# A - Bigrams

> 关键词：签到

## 思路

对于所有的字母种类，只要有两种字母的数量 $\ge 2$ 即可。也就是 $sum - n \ge 2$ 符合条件。

## Code

```c++

// URL: https://codeforces.com/contest/2242/problem/A

#include <bits/stdc++.h>

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
```

# B - Predominant Frequency Division 

> 关键词：枚举，后缀处理

## 思路

要把数组切两刀，分成三段。考虑枚举第一刀，快速判断第二刀是否存在。

左段直接使用前缀和判断即可。对于中段，可以定义 $s[i] = s1[i] + s2[i] - s3[i]$，中段合法等价于 $s[j]-s[i]\ge0$，即 $s[j] \ge s[i]$。右段普遍成立，只需要非空，也就是第二刀不能在 $n$ 位置切。

用一个后缀最大值数组，快速判断是否存在合法的第二刀，$suf[i]=\max_{i\le k\le n-1} s[k]$。注意最大值只取到 $n-1$，因为第二刀不能在 $n$ 位置切。

然后枚举第一刀的位置 $i$，先判断左段是否合法，再判断是否存在合法的第二刀。如果存在就输出 $YES$。

## Code

```c++
// Problem: CF 2242 B
// Contest: Codeforces - Educational Codeforces Round 192 (Rated for Div. 2)
// URL: https://codeforces.com/contest/2242/problem/B
// Time: 2026-07-06 22:41:33
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
#define eb emplace_back
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

void solve()
{
    int n;
    cin >> n;

    vector<int> a(n + 1), s1(n + 1), s2(n + 1), s3(n + 1);
    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];

        s1[i] = s1[i - 1];
        s2[i] = s2[i - 1];
        s3[i] = s3[i - 1];

        if (a[i] == 1)
            s1[i]++;
        else if (a[i] == 2)
            s2[i]++;
        else
            s3[i]++;
    }

    if (n < 3)
    {
        cout << "NO" << endl;
        return;
    }

    vector<int> s(n + 1), suf(n + 2, -inf);
    for (int i = 1; i <= n; i++)
        s[i] = s1[i] + s2[i] - s3[i];

    for (int i = n - 1; i >= 1; i--)
        suf[i] = max(suf[i + 1], s[i]);

    for (int i = 1; i < n - 1; i++)
    {
        bool flag = (s1[i] >= s2[i] + s3[i]);

        if (!flag)
            continue;

        if (suf[i + 1] >= s[i])
        {
            cout << "YES" << endl;
            return;
        }
    }

    cout << "NO" << endl;
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}

```
