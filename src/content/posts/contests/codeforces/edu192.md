# A - Bigrams

> 关键词：签到

## 思路

对于所有的字母种类，只要有两种字母的数量 $\ge 2$ 即可。也就是 $sum - n \ge 2$ 符合条件。

## Code

```c++
const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
```

# B - Predominant Frequency Division 

> 关键词：枚举，后缀处理

## 思路

要把数组切两刀，分成三段。考虑枚举第一刀，快速判断第二刀是否存在。

左段直接使用前缀和判断即可。对于中段，可以定义 $s[i] = s1[i] + s2[i] - s3[i]$，中段合法等价于 $s[j]-s[i]\ge0$，即 $s[j] \ge s[i]$。右段普遍成立，只需要非空，也就是第二刀不能在 $n$ 位置切。

用一个后缀最大值数组，快速判断是否存在合法的第二刀，$suf[i]=\max_{i\le k\le n-1} s[k]$。注意最大值只取到 $n-1$，因为第二刀不能在 $n$ 位置切。

然后枚举第一刀的位置 $i$，先判断左段是否合法，再判断是否存在合法的第二刀。如果存在就输出 $YES$。

## Code

```c++
// Problem: CF 2242 B
// Contest: Codeforces - Educational Codeforces Round 192 (Rated for Div. 2)
// URL: https://codeforces.com/contest/2242/problem/B
// Time: 2026-07-06 22:41:33
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
#define eb emplace_back
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

void solve()
{
    int n;
    cin >> n;

    vector<int> a(n + 1), s1(n + 1), s2(n + 1), s3(n + 1);
    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];

        s1[i] = s1[i - 1];
        s2[i] = s2[i - 1];
        s3[i] = s3[i - 1];

        if (a[i] == 1)
            s1[i]++;
        else if (a[i] == 2)
            s2[i]++;
        else
            s3[i]++;
    }

    if (n < 3)
    {
        cout << "NO" << endl;
        return;
    }

    vector<int> s(n + 1), suf(n + 2, -inf);
    for (int i = 1; i <= n; i++)
        s[i] = s1[i] + s2[i] - s3[i];

    for (int i = n - 1; i >= 1; i--)
        suf[i] = max(suf[i + 1], s[i]);

    for (int i = 1; i < n - 1; i++)
    {
        bool flag = (s1[i] >= s2[i] + s3[i]);

        if (!flag)
            continue;

        if (suf[i + 1] >= s[i])
        {
            cout << "YES" << endl;
            return;
        }
    }

    cout << "NO" << endl;
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}

```

# C - Unstable Elements

> 关键词：思维

## 思路

由于数组已经排好序，可以把整个数组压缩成若干个块 + 块的长度。例如 $[1,1,1,2,4,4,5]$，看作 $1^3,\ 2^1,\ 4^2,\ 5^1$，对应块长度为 $[3,1,2,1]$，每一次操作会标记每一块的第一个元素。

如果删除所有标记元素，每一块都会少一个；如果复制所有标记元素，每一块都会多一个。

所以这道题可以转化为：有若干个块，每次可以让所有还存在的块长度同时 $+1$ 或 $-1$。长度变成 $0$ 的块永久消失。问最终总长度为 $k$ 的不同数组有多少种。

令当前净变化量 $d$ 为复制次数减去删除次数，如果这个块一直没死，它的当前长度就是 $len+d$，但如果某一时刻 $len+d=0$，这个块就会直接消失。假设整个操作过程中，最多曾经净删除了 $q$ 次，那么初始长度 $\le q$ 的块一定会死，而最后活着的块一定满足 $len > q$。

### 代码思路

1. `cnt[x]` 统计数字 $x$ 出现次数，也就是块长度；
2. `cnts[len]` 统计长度为 $len$ 的块有几个；
3. 后缀和后，`cnts[i]` 表示长度至少为 $i$ 的块有几个；
4. 用 `t` 表示已经删除了多少层；
5. 用 `n` 表示当前数组长度；
6. 用 `num` 表示当前活着的块数；
7. 当前状态能通过复制到达长度 $k$，当且仅当：$k\ge n$，$(k-n)\bmod num=0$
8. 用 `vis[num]` 避免同一批活块重复计数。

## Code

```c++
// Problem: CF 2242 C
// Contest: Codeforces - Educational Codeforces Round 192 (Rated for Div. 2)
// URL: https://codeforces.com/contest/2242/problem/C
// Time: 2026-07-06 23:12:08
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
#define eb emplace_back
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

void solve()
{
    int n, k;
    cin >> n >> k;

    vector<int> a(n + 1), len(n + 2), cnts(n + 2);

    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];
        len[a[i]]++;
    }

    // num为当前还存活的块数
    int num = 0;
    for (int i = 1; i <= n; i++)
    {
        if (len[i])
        {
            num++;
            cnts[len[i]]++;
        }
    }

    // 后缀和
    for (int i = n - 1; i > 0; i--)
        cnts[i] = cnts[i + 1] + cnts[i];

    // t 为删除层数，ans为可达长度为k的不同数组数量
    int t = 0, ans = 0;
    vector<int> vis(num + 1); // 避免同一个数量重复计数

    while (num && n > 0) // 还有活块且当前长度非空就继续模拟
    {
        if (k >= n && (k - n) % num == 0 && !vis[num])
        {
            vis[num] = 1;
            ans++;
        }

        t++;
        n -= num;
        num = cnts[t + 1]; // 长度至少为 t + 1 的块数
    }
    cout << ans << endl;
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}

```

# D - Two Digit Strings

> 关键词：DP，前缀和

## 思路

注意到操作等价于分段求和再模 $10$，每个最终字符本质上就是某个连续段的数位和模上 $10$。

定义前缀和 $pre1[i]=a_1+a_2+\cdots+a_i \pmod{10}$，$pre2[j]=b_1+b_2+\cdots+b_j \pmod{10}$。

假设 $a$ 和 $b$ 的某一段要对应相等。例如 $a[l_1,r_1]$ 和 $b[l_2,r_2]$，需要满足 $sum(a[l_1,r_1])\equiv sum(b[l_2,r_2])\pmod{10}$。

如果上一组切点是 $(i',j')$，当前切点是 $(i,j)$。那么当前段分别是 $a[i'+1,i]$，$b[j'+1,j]$。

要求 $pre1[i]-pre1[i']\equiv pre2[j]-pre2[j']\pmod{10}$，移项得到 $pre1[i]-pre2[j]\equiv pre1[i']-pre2[j']\pmod{10}$。

由于初始切点是 $(0,0)$，且 $pre1[0]=pre2[0]=0$，即 $pre1[0]-pre2[0]=0$。

**因此所有合法切点都必须满足 $pre1[i]=pre2[j]$，所以只需要匹配相等的前缀和。 **

最终两个字符串要完全相等，那么它们所有字符的和模 $10$ 也必须相等，而无论怎么合并，相邻两数变成它们的和模 $10$，整个字符串的总和模 $10$ 是不变的。所以必须满足 $pre1[n]=pre2[m]$，如果不满足就无解。

定义 $dp[i][j]$ 表示在 $a$ 的前 $i$ 个位置和 $b$ 的前 $j$ 个位置中，最多可以选出多少对合法切点，即满足 $pre1[i]=pre2[j]$ 的点。

这个 $dp$ 本质上就是对两个前缀和序列做 $LCS$。最终得到一个长度为 $L$ 的字符串，等价于把原串切成 $L$ 段，并且每一段压缩后相等。所以我们要找出最多能切出多少对合法段。

## Code

```c++
// Problem: CF 2242 D
// Contest: Codeforces - Educational Codeforces Round 192 (Rated for Div. 2)
// URL: https://codeforces.com/contest/2242/problem/D
// Time: 2026-07-07 14:10:58
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
#define eb emplace_back
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

void solve()
{
    string a, b;
    cin >> a >> b;

    int n = a.size(), m = b.size();
    a = " " + a, b = " " + b;

    vector<ll> pre1(n + 1), pre2(m + 1);

    for (int i = 1; i <= n; i++)
        pre1[i] = (pre1[i - 1] + (a[i] - '0')) % 10;
    for (int i = 1; i <= m; i++)
        pre2[i] = (pre2[i - 1] + (b[i] - '0')) % 10;

    if (pre1[n] != pre2[m])
    {
        cout << -1 << endl;
        return;
    }

    vector<vector<ll>> dp(n + 1, vector<ll>(m + 1));

    for (int i = 1; i <= n; i++)
    {
        for (int j = 1; j <= m; j++)
        {
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
            if (pre1[i] == pre2[j])
                dp[i][j] = max(dp[i][j], dp[i - 1][j - 1] + 1);
        }
    }

    cout << dp[n][m] << endl;
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}
```
