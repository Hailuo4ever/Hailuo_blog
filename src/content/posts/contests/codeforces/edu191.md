# B - Different Distances

> 关键词：构造

## 思路

题目要求每个数字出现 $4$ 次，并且同一个数字的三段相邻出现距离两两不同。

我们考虑循环移位，每输出一段后，分别将当前排列循环左移 $1,2,3$ 位，再输出下一段。

严格一点的证明过程如下：

对于任意数字 $x$，考虑它在相邻两段中的位置差。若两段之间循环左移了 $k$ 位，则对应的出现距离模 $n$ 为：$-k \pmod n$，当 $n \ge 3$ 时，三次位移量分别为：$1,2,3$，它们模 $n$ 两两不同，因此三段出现距离也必然两两不同。

特别地，当 $n=2$ 时，构造得到：$[1,2,\ 2,1,\ 2,1,\ 1,2]$，同样满足要求。因此，该构造对所有合法的 $n$ 均成立。

# C - Cost of a Branket Sequence

> 关键词：枚举，思维

## 思路

因为一个合法括号序列的长度一定是偶数，所以可以把问题转化成：最小化剩余字符串中最多能匹配出的括号对数。

如果最多能匹配出 $m$ 对括号，代价为 $2m$。

考虑代价在什么时候可以取到最小值 $0$。如果一个字符串中存在 `(` 在 `)` 的左边，至少可以选出一个子序列 `()`，此时代价至少为 $2$。因此代价为 $0$ 等价于字符串中不存在 `(` 在 `)` 的左边。

考虑如何将字符串的代价变为 $0$。我们枚举一个分界点 $i$，删除分界点左边的所有 `(`，右边的所有 `)`。

设 $pre_{L_i}$ 为前 $i$ 个字符中 `(` 的数量，$pre_{r_i}$ 为前 $i$ 个字符中 `)` 的数量，设总共右括号数量为 $totR$，如果分界线在 $i$ 后面，那么需要删除的数量为左边的 `(` 加上右边的 `)`，即 $pre_{L_i}+(totR-pre_{R_i})$。

我们要找所有分界点的最小值，即 $M=\min_{0\le i\le n}\left(preL_i+totR-preR_i\right)$。这个 $M$ 就是最小删除的数量，**也是原串最多能匹配出的括号对数。**

## Code

```c++
// Problem: CF 2233 C
// Contest: Codeforces - Educational Codeforces Round 191 (Rated for Div. 2)
// URL: https://codeforces.com/contest/2233/problem/C
// Time: 2026-06-19 16:26:03
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const int N = 0;

void solve()
{
    int n, k;
    string s;

    cin >> n >> k >> s;

    vector<int> preL(n + 1, 0), preR(n + 1, 0);

    for (int i = 0; i < n; i++)
    {
        preL[i + 1] = preL[i] + (s[i] == '(');
        preR[i + 1] = preR[i] + (s[i] == ')');
    }

    int totR = preR[n], best = inf, pos = 0;

    for (int i = 0; i <= n; i++)
    {
        int t = preL[i] + (totR - preR[i]);

        if (t < best)
            best = t, pos = i;
    }

    string res(n, '0');
    int cnt = min(k, best);

    for (int i = 0; i < pos && cnt > 0; i++)
    {
        if (s[i] == '(')
            res[i] = '1', cnt--;
    }

    for (int i = pos; i < n && cnt > 0; i++)
    {
        if (s[i] == ')')
            res[i] = '1', cnt--;
    }

    cout << res << endl;
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}
```
