# B - Different Distances

> 关键词：构造

## 思路

题目要求每个数字出现 $4$ 次，并且同一个数字的三段相邻出现距离两两不同。

我们考虑循环移位，每输出一段后，分别将当前排列循环左移 $1,2,3$ 位，再输出下一段。

严格一点的证明过程如下：

对于任意数字 $x$，考虑它在相邻两段中的位置差。若两段之间循环左移了 $k$ 位，则对应的出现距离模 $n$ 为：$-k \pmod n$，当 $n \ge 3$ 时，三次位移量分别为：$1,2,3$，它们模 $n$ 两两不同，因此三段出现距离也必然两两不同。

特别地，当 $n=2$ 时，构造得到：$[1,2,\ 2,1,\ 2,1,\ 1,2]$，同样满足要求。因此，该构造对所有合法的 $n$ 均成立。

## Code

```c++
// Problem: CF 2233 B
// Contest: Codeforces - Educational Codeforces Round 191 (Rated for Div. 2)
// URL: https://codeforces.com/contest/2233/problem/B
// Time: 2026-06-10 00:57:04
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const int N = 0;

void solve()
{
    int n;
    cin >> n;

    vector<int> a(n);
    iota(all(a), 1);

    for (int i = 1; i <= 4; i++)
    {
        for (auto x: a)
            cout << x << " ";
        rotate(a.begin(), a.begin() + (i % n), a.end());
    }
    cout << endl;
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}
```
