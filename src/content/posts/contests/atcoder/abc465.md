# C - Reverse Permutation

> 关键词：思维，双端队列

## 思路

首先注意到在处理第 $k$ 次操作时，第 $k$ 个位置后面的所有数不受影响。因此可以把操作过程看作：每次把新数字 $k$ 加入当前序列，然后根据 $S_k$ 决定是否整体反转当前序列。

使用双端队列维护答案序列，如果不反转前 $k$ 项，相当于把 $k$ 放到答案序列末尾，否则放在开头。当操作为 $o$ 时更新反转标记。

## Code

```c++
// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()

// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};

void solve()
{
    int n;
    string s;

    // vector<int> res;
    deque<int> res;
    bool flag = false;

            res.push_front(i);
        else
            res.push_back(i);

    }

    if (!flag)
        for (auto it = res.rbegin(); it != res.rend(); ++it)
            cout << *it << ' ';
    else
        for (auto i: res)
            cout << i << ' ';
}

        solve();
```

# D - X to Y

> 关键词：LCA

## 思路

从 $x$ 可以走到父亲节点 $\lfloor x / K \rfloor$，也可以从 $x$ 走到任意一个儿子节点 $y$，满足：$Kx \le y \le Kx + K - 1$。

所以每个数 $v$ 的父亲是 $fa(v)=\left\lfloor \frac{v}{K} \right\rfloor$，这是一棵以 $0$ 为根的树。

从 $X$ 开始一直除以 $K$，记录它到每个祖先的距离。然后从 $Y$ 开始不断除以 $K$，第一个遇到的在 $X$ 祖先链里的点，就是最近公共祖先。答案即为 $X$ 到 $LCA$ 的距离加上 $Y$ 到 $LCA$ 的距离。

注：$X$ 变小路径唯一，$Y$ 变小路径也唯一，两个第一次相遇即为答案。

## Code

```c++
// Problem: AT ABC465 D
// Contest: AtCoder - AtCoder Beginner Contest 465
// URL: https://atcoder.jp/contests/abc465/tasks/abc465_d
// Time: 2026-07-04 20:46:05
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

void solve()
{
    ll X, Y, k;
    cin >> X >> Y >> k;

    unordered_map<ll, ll> dist;

    ll x = X, y = Y;
    int dx = 0, dy = 0;

    while (true)
    {
        dist[x] = dx;

        if (x == 0)
            break;

        x /= k;
        dx++;
    }

    while (true)
    {
        if (dist.count(y))
        {
            cout << dist[y] + dy << endl;
            return;
        }

        y /= k, dy++;
    }
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}
```
