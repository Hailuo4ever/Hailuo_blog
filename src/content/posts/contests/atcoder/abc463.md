# C - Tallest at the Moment

> 关键词：后缀处理

## 思路

注意到输入的时间是递增的，所以方便了一点。

我们从后往前维护最大身高，由于输入已经按照 $L_i$ 从小到大排序，从后往前遍历时，当前看到的人满足 $L_i$ 大于等于当前离开时间。使用 `map` 存储某个离开时间对应的答案。

查询时使用 `upper_bound` 查找比查询时间大的第一个位置即可，由于题目保证起码有一个人，不用判边界。

## Code

```c++

#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

void solve()
{
    int n;
    cin >> n;

    vector<ll> h(n), l(n);
    for (int i = 0; i < n; i++)
        cin >> h[i] >> l[i];

    map<ll, ll> mp;

    ll mx = 0;
    for (int i = n - 1; i >= 0; i--)
    {
        mx = max(mx, h[i]);
        mp[l[i]] = mx;
    }

    int q;
    cin >> q;

    while (q--)
    {
        ll t;
        cin >> t;

        auto it = mp.upper_bound(t);
        cout << it->second << endl;
    }
}

int main()
{
    fastio();

    int T = 1;
    // cin >> T;

    while (T--)
        solve();

    return 0;
}
```

# D - Maximize the Gap

> 关键词：二分答案，贪心

## 思路

每块布是一个闭区间 $[L_i,R_i]$，两个区间不重叠的条件是 $R_i<L_j$，且距离为 $L_j-R_i$。

假设我们选出的区间按照位置排序为：$I_1,I_2,\cdots ,I_k$，那么分数为所有区间两两距离的最小值，容易发现，只需要检查每个相邻的被选区间的间隔，取最小值即可。

可以二分一个距离 $d$，判断是否能选出 $K$ 个区间，使得相邻被选区间之间的距离至少为 $d$。

也就是对于连续选择的两个区间 $a,b$，必须满足 $L_b - R_a \ge d$。

由于 $d$ 越大越难选，$d$ 越小越容易选，所以答案具有单调性。当固定了 $d$ 后，策略如下：

1. 按右端点 $R_i$ 从小到大排序。
2. 每次选择当前能选的、右端点最小的区间。
3. 记录上一个被选区间的右端点 $last$。
4. 当前区间 $[L_i, R_i]$ 可以选，当且仅当：$L_i - last \ge d$。

关于无解情况，题目要求选出的 $K$ 块布两两不重叠。

因为所有端点都是整数，且区间是闭区间，所以两个区间不重叠至少需要 $L_b - R_a \ge 1$，因此检查 $d=1$ 能否选出 $k$ 个区间，如果不能就输出 $-1$。

## Code

```c++
// Problem: AT ABC463 D
// Contest: AtCoder - AtCoder Beginner Contest 463
// URL: https://atcoder.jp/contests/abc463/tasks/abc463_d
// Time: 2026-06-20 20:00:07
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 0;

bool cmp(pll &a, pll &b)
{
    if (a.second != b.second)
        return a.second < b.second;
    return a.first < b.first;
}

void solve()
{
    int n, k;
    cin >> n >> k;

    vector<pll> seg(n);
    for (int i = 0; i < n; i++)
    {
        ll l, r;
        cin >> l >> r;
        seg[i] = {l, r};
    }

    sort(all(seg), cmp);

    auto check = [&](ll d) -> bool
    {
        int cnt = 0;
        ll lst = -INF;

        for (auto [l, r]: seg)
        {
            if (l >= lst + d)
            {
                cnt++;
                lst = r;
            }
        }

        return cnt >= k;
    };

    if (!check(1))
    {
        cout << -1 << endl;
        return;
    }

    ll l = 1, r = 1e9 + 10;

    while (l + 1 < r)
    {
        ll mid = (l + r) >> 1;
        if (check(mid))
            l = mid;
        else
            r = mid;
    }

    cout << l << endl;
}

int main()
{
    fastio();

    int T = 1;
    // cin >> T;

    while (T--)
        solve();

    return 0;
}
```

# E - Roads and Gates

> 关键词：图论建模，虚拟源点

## 思路

对于传送门的条件，我们不能朴素建图，这样就有 $O(N^2)$ 条边，无法接受。

观察传送代价 $X_i+X_j+Y$​，它可以被拆成三段，也就是说，从城市 $i$ 传送到城市 $j$，可以理解为：

1. 从城市 $i$ 进入传送网络，花费 $X_i$；
2. 在传送网络内部移动，花费 $Y$；
3. 从传送网络出来到城市 $j$，花费 $X_j$。

这就是虚拟点建图的突破口。对于每一个城市 $i$，都建两条**到虚拟源点的有向边** $i \to G,\quad \text{cost}=X_i$，$G \to i,\quad \text{cost}=X_i+Y$

> [!NOTE]
>
> 总结：虚拟源点建图的本质是利用了传送代价的可拆分性，将 $O(N^2)$ 条传送边压缩到 $O(N)$。

## Code

```c++
// Problem: AT ABC463 E
// Contest: AtCoder - AtCoder Beginner Contest 463
// URL: https://atcoder.jp/contests/abc463/tasks/abc463_e
// Time: 2026-06-20 20:00:08
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;
using p = array<ll, 3>;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const ll INF = 4e18;
const int N = 2e5 + 10;

ll dist[N];
bool st[N];

struct Edge
{
    int v;
    ll w;
};

void solve()
{
    fill(begin(dist), end(dist), INF);
    fill(begin(st), end(st), false);

    ll n, m, y;
    cin >> n >> m >> y;

    vector<vector<Edge>> adj(n + 2);
    for (int i = 0, u, v; i < m; i++)
    {
        ll w;
        cin >> u >> v >> w;

        adj[u].emplace_back(v, w);
        adj[v].emplace_back(u, w);
    }

    for (int i = 1; i <= n; i++)
    {
        ll x;
        cin >> x;

        adj[i].emplace_back(n + 1, x + y);
        adj[n + 1].emplace_back(i, x);
    }

    priority_queue<pll, vector<pll>, greater<pll>> pq;
    pq.push({0, 1});

    while (!pq.empty())
    {
        auto [d, u] = pq.top();
        pq.pop();

        if (st[u])
            continue;

        st[u] = true;

        for (auto &[v, w]: adj[u])
        {
            if (dist[v] > d + w)
            {
                dist[v] = d + w;
                pq.push({dist[v], v});
            }
        }
    }

    for (int i = 2; i <= n; i++)
        cout << dist[i] << " ";
}

int main()
{
    fastio();

    int T = 1;
    // cin >> T;

    while (T--)
        solve();

    return 0;
}
```
