# C - Not Covered Points

> 关键词：数学

## 思路

按照 $x$ 排序后可以消掉一维，接着维护全局出现的 $y$ 坐标最小值即可。

## Code

```c++
// Problem: AT ABC462 C
// Contest: AtCoder - CodeQUEEN 2026 -qual- (AtCoder Beginner Contest 462)
// URL: https://atcoder.jp/contests/abc462/tasks/abc462_c
// Time: 2026-06-13 20:00:14
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const int N = 0;

void solve()
{
    int n;
    cin >> n;

    vector<pii> points(n);
    for (int i = 0; i < n; i++)
    {
        int x, y;
        cin >> x >> y;
        points[i] = {x, y};
    }

    sort(all(points));

    int minn = inf, res = 0;
    for (int i = 0; i < n; i++)
    {
        if (points[i].second < minn)
        {
            minn = points[i].second;
            res++;
        }
    }
    cout << res << endl;
}

int main()
{
    fastio();

    int T = 1;
    // cin >> T;

    while (T--)
        solve();

    return 0;
}

```

# D - Accomplice

> 关键词：差分

## 思路

将每个人看作一个可行的时间区间，第 $i$ 个人可以参与犯罪的开始时间范围是：$[S_i, T_i - D]$。

相比于枚举一对犯人，计算他们有多少个合法开始时间，我们枚举一个开始时间，计算有多少对犯人可以在此时作案。

由于时间范围是 $10^6$，我们使用全局差分数组来维护整个时间轴上的犯人信息，求一次前缀和就能还原。

对于每个位置，答案为 $C_{cnt}^2$，累加所有位置的答案即可。

## Code

```c++
// Problem: AT ABC462 D
// Contest: AtCoder - CodeQUEEN 2026 -qual- (AtCoder Beginner Contest 462)
// URL: https://atcoder.jp/contests/abc462/tasks/abc462_d
// Time: 2026-06-13 20:00:15
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const int inf = 0x3f3f3f3f;
const int N = 1e6 + 10;

ll a[N];

void debug()
{
    for (int i = 1; i <= 20; i++)
        cout << a[i] << " ";
    cout << endl;
}

void solve()
{
    ll n, d;
    cin >> n >> d;

    for (int i = 1; i <= n; i++)
    {
        ll s, t;
        cin >> s >> t;

        if (s > t - d)
            continue;

        a[s]++, a[t - d + 1]--;
    }

    ll res = 0;
    for (int i = 1; i < N; i++)
        a[i] += a[i - 1];

    // debug();

    for (int i = 1; i < N; i++)
        res += a[i] * (a[i] - 1) / 2;

    cout << res << endl;
}

int main()
{
    fastio();

    int T = 1;
    // cin >> T;

    while (T--)
        solve();

    return 0;
}

```

# E - Alternating Costs

> 关键词：数学

## 思路

> 这道题不能只考虑最短路径。
>
> 通常从 $(0,0)$ 到 $(X,Y)$ 至少需要 $|X|+|Y|$ 步。但当 $A$ 和 $B$ 差距较大时，绕路反而可能更便宜。
>
> 例如：$A=1,\quad B=100,\quad (X,Y)=(2,0)$
>
> 直接横向移动两次，费用为：$A+B=101$，但可以走：$(0,0)\to(1,0)\to(1,1)\to(2,1)\to(2,0)$
>
> 每一步的费用都是 $1$，总费用仅为 $4$，因此，需要考虑可能的绕路。

首先可以忽略坐标的正负，直接取绝对值。

其次，可以将相邻的两步绑定为一组，设：$H$ 表示横向移动，$V$ 表示纵向移动。

共有四种类型：

| 两步的方向 | 费用  | 效果                    |
| ---------- | ----- | ----------------------- |
| `H, H`     | $A+B$ | 横向移动 $0$ 或 $2$ 格  |
| `V, V`     | $A+B$ | 纵向移动 $0$ 或 $2$ 格  |
| `H, V`     | $2A$  | 横向和纵向各移动 $1$ 格 |
| `V, H`     | $2B$  | 横向和纵向各移动 $1$ 格 |

令 $m=\min(A,B)$，对于横向和纵向各移动 $1$ 格的情况，可以选择更便宜的顺序，因此最低费用为 $2m$。

假设我们需要横向移动两格，可以直接使用 $H,H$，代价 $A+B$，也可以绕路，用两个斜向，代价 $4m$，因此同一方向移动两个的最低费用为：$C=\min(A+B,4m)$，纵向移动两格也完全相同。

考虑分奇偶讨论。定义 $F(u,v)$ 表示使用偶数步，移动到终点的最低费用，**只有当 $u+v$ 为偶数时才会用这个函数计算。**

先同时处理两个方向的移动，再处理剩余部分，最终得到：$\boxed{
F(u,v)=2m\cdot \min(u,v)+C\cdot \frac{|u-v|}{2}
}$

其中 $m=min(A, B)$，$C=min(A+B,4m)$。

如果 $u+v$ 为奇数，答案为：$\min
\begin{cases}
A+F(|u-1|,v)\\
A+F(u+1,v)\\
B+F(u,|v-1|)\\
B+F(u,v+1)
\end{cases}$

## Code

```c++
// Problem: AT ABC462 E
// Contest: AtCoder - CodeQUEEN 2026 -qual- (AtCoder Beginner Contest 462)
// URL: https://atcoder.jp/contests/abc462/tasks/abc462_e
// Time: 2026-06-13 20:00:16
#include <bits/stdc++.h>
using namespace std;

// clang-format off
#define endl '\n'
#define all(x) (x).begin(), (x).end()
#define fastio() ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
// clang-format on

using ll = long long;
using ull = unsigned long long;
using pii = pair<int, int>;
using pdd = pair<double, double>;
using pll = pair<long long, long long>;
using i128 = __int128;

const int dx[] = {-1, 0, 1, 0, -1, 1, 1, -1};
const int dy[] = {0, 1, 0, -1, 1, 1, -1, -1};
const ll inf = 4e18;
const int N = 0;

void solve()
{
    int a, b, x, y;
    cin >> a >> b >> x >> y;

    x = abs(x), y = abs(y);

    int t = min(x, y);
    ll res = 2ll * t * min(a, b);
    x -= t;
    y -= t;

    auto f = [&](int l, int a, int b) -> ll
    {
        ll res = 1ll * (l + 1) / 2 * a + 1ll * l / 2 * b;
        res = min(res, l / 2 * 4ll * a + l % 2 * a);
        res = min(res, l / 2 * 4ll * b + l % 2 * 3ll * b);
        return res;
    };

    if (x)
        res += f(x, a, b);
    else if (y)
        res += f(y, b, a);

    cout << res << '\n';
}

int main()
{
    fastio();

    int T = 1;
    cin >> T;

    while (T--)
        solve();

    return 0;
}

```
